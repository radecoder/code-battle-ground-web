Some font properties:

    1. Line-height : it increase the height (gap) between lines
    2. text-transform: capitalize; this is how to capitalize and smalize the letters
    3. text-decoration: underline;
    4. text-decoration-color: red;
    5. text-indent: 100px; on increasing, it goes left -> Right
    6. Text overflow
